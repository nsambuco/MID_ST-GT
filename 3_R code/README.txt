In this folder we share the R codes relative to the Clustering analyses:
- "Hierarchical_Kmeans_DISCOVERY.R": R code for the implementation of the clustering on Discovery dataset 
- "Hieratchical_Kmeans_EXTERNAL_DATASETS.R": R code for the generalization of the clustering in other external datasets
- "Leave_Site_Out_Validation_DISCOVERY.R": R code for the validation of the clustering algorithm in a multi-site dataset
- Two .xlsx file: one corresponding to the input file for performing the algorithm in the Discovery ("dataset_clustering.xlsx") and for the Leave-site-out, and the other as input file for running the algorithm in external datasets.
